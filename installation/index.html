    Installation

    You can install the package via pip:

    pip install aws_resource_validator

    Requirements

    • Python 3.8 or later
    • pip

    Setup

    1. Install the package:
    pip install aws_resource_validator
    1. Verify the installation:
    python -m aws_resource_validator
