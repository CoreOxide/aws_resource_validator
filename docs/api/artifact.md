# Artifact Service

### LongStringAttribute
- **Type**: string
- **Pattern**: `^[^<>]*$`
- **Min Length**: 1
- **Max Length**: 1024

### ReportId
- **Type**: string
- **Pattern**: `^report-[a-zA-Z0-9]{16}$`

### ShortStringAttribute
- **Type**: string
- **Pattern**: `^[a-zA-Z0-9_\-\s]*$`
- **Min Length**: 1
- **Max Length**: 256
