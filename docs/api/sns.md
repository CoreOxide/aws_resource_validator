# Sns Service

### Iso2CountryCode
- **Type**: string
- **Pattern**: `^[A-Za-z]{2}$`
- **Max Length**: 2

### OTPCode
- **Type**: string
- **Pattern**: `^[0-9]+$`
- **Min Length**: 5
- **Max Length**: 8

### PhoneNumberString
- **Type**: string
- **Pattern**: `^(\+[0-9]{8,}|[0-9]{0,9})$`
- **Max Length**: 20
